### Тесты

Запуск тестов через Makefile:

```bash
# Запустить все тесты
make test

# Запустить тесты с отчётом о покрытии (выводит общий процент)
make test-cover

# Запустить тесты, сгенерировать HTML-отчёт и открыть его в браузере
make test-cover-html

# Очистить файлы отчётов
make clean-test
```
